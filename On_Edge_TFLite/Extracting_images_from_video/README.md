# Video to Frames Extraction

This Python script extracts frames from a video file and saves them as individual images in a specified folder. It is useful for computer vision projects, video analysis, or preprocessing videos for machine learning tasks.

---

## Features

- Reads a video file using OpenCV.
- Extracts each frame sequentially.
- Saves frames as PNG images in a designated folder.
- Measures and prints the total execution time.

---

## Requirements

- Python 3.x
- OpenCV
- OS module (standard with Python)
- Time module (standard with Python)

You can install OpenCV using pip:

```bash
pip install opencv-python
```
